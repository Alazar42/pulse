# Contributor Covenant Code of Conduct

Our community is dedicated to providing a welcoming and respectful environment
for all contributors. By participating in this project, you agree to abide by
this Code of Conduct.

Be respectful. Harassment, hateful language, or personal attacks are not
allowed. Treat others with empathy and consideration.

If you witness or are subject to unacceptable behavior, please contact the
project maintainers with as much detail as possible.

This project follows the Contributor Covenant; see
https://www.contributor-covenant.org/ for the full policy.
